## Project description
This project is an audit of the open source software Git. It covers its origin, licensing model, ethical considerations, Linux footprint, ecosystem, and comparison with proprietary software.

The project also  includes five shell scripts demonstrating practical Linux and shell scripting concepts.

## Scripts Overview

### Script 1: System Identity Report
Displays system information such as kernel version, user, uptime, date and operating system details.

### Script 2: FOSS Package Inspector
Checks whether a package is installed and displays its version and a short description.

### Script 3: Disk and Permissino Auditor
Analyzes system directories and displays permissions, ownership, and disk usage.

### Script 4: Log File Analyzer
Reads a file, counts occurences of a keyword, and shows the last matching lines.

### Script 5: OPen Source Manifesto Generator
Generates a personalized open-source manifesto based on user input and saves it to a file.

## How to run the scripts

1. Open Git Bash or terminal
2. Navigate to the folder:
   cd oss-audit
3. Give execution Permission:
   chmod +x script_name.sh
4. Run the script:
   ./script_name.sh
